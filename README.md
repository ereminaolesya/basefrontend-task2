https://ereminaolesya.github.io/basefrontend-task2/

## Описание

Реализация HTML+CSS проекта рекламного баннера с использованием линейных градиентов для создания визуально привлекательных фоновых и текстовых эффектов.

Задача взята из учебного практического курса.

## Требования к задаче

Обязательное использование линейных градиентов:
Фоновый градиент для основного контейнера рекламного блока
Текстовый градиент для заголовка
Градиентная кнопка “Get started”
